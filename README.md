## Setup

install packages

```Shell
yarn
```

start

```Shell
yarn start
```

## Examples

> type "help" in tabash for more infos

list registered links

```Shell
ls
```

register link

```Shell
gh=github.com
```

open link (github)

```Shell
gh
```

install package from [tire](https://github.com/jshan2017/tire) (alarm)

```Shell
tire install alarm
```
